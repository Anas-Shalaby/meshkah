# 📚 نظام ختمات الكتب - التحديث الجديد

## نظرة عامة
تم تطوير نظام ختمات الكتب في تطبيق **مشكاة** ليشمل ثلاث ميزات جديدة تعزز تجربة القراءة وتزيد من الالتزام والتحفيز.

---

## 🎯 الميزات الجديدة

### 1. نظام التعهد (Pledge System)
عندما تبدأ ختمة جديدة، يمكنك الآن كتابة **نيتك وتعهدك** لإتمام الكتاب.

**كيف يعمل؟**
- عند الضغط على "ابدأ الختمة" يظهر Modal جميل
- تختار عدد الأحاديث اليومية
- تكتب تعهدك/نيتك (اختياري لكن محفز!)
- التعهد يظهر لك يومياً كتذكير قبل القراءة

**الفائدة:**
- تجديد النية يومياً
- الالتزام بالهدف
- زيادة الدافع للاستمرار

---

### 2. نظام الرفقة (Buddy System)
اختر **رفيق ختمة** من أصدقائك ليقرأ معك نفس الكتاب!

**كيف يعمل؟**
- في تبويب "الأصدقاء" ترى من يقرأ نفس الكتاب
- اضغط زر "رفقة" 💜 لإرسال طلب
- صديقك يقبل أو يرفض الطلب
- بعد القبول، تصبحون رفيقين!

**ماذا يمكنك فعله مع رفيقك؟**
- رؤية تقدمه في الختمة
- إرسال رسائل تشجيع سريعة
- تلقي إشعارات عند تقدمه

**الفائدة:**
- المنافسة الإيجابية
- التشجيع المتبادل
- لا تشعر بأنك وحدك!

---

### 3. تقويم التقدم (Progress Calendar)
شاهد تقدمك الشهري في **تقويم ملون** جميل!

**كيف يعمل؟**
- في تبويب "التقدم" ترى تقويم الشهر
- كل يوم ملون حسب الإنجاز:
  - 🟢 أخضر = أكملت الورد
  - 🟡 أصفر = قرأت جزء
  - ⬜ رمادي = لم تقرأ
- إحصائيات شهرية (أيام القراءة، المتوسط، أفضل streak)

**الفائدة:**
- رؤية واضحة لنمط القراءة
- تحفيز لملء التقويم بالأخضر
- تتبع الإنجاز على مستوى الشهر

---

## 🛠️ التغييرات التقنية

### قاعدة البيانات
```sql
-- جدول الرفقاء
CREATE TABLE journey_buddies (
  id INT PRIMARY KEY,
  journey_id INT,
  buddy_journey_id INT,
  status ENUM('pending', 'accepted', 'declined'),
  created_at TIMESTAMP
);

-- جدول رسائل التشجيع
CREATE TABLE buddy_messages (
  id INT PRIMARY KEY,
  from_journey_id INT,
  to_journey_id INT,
  message TEXT,
  is_read BOOLEAN
);

-- أعمدة جديدة في book_journeys
ALTER TABLE book_journeys ADD COLUMN pledge TEXT;
ALTER TABLE book_journeys ADD COLUMN pledge_shared BOOLEAN;
```

### الملفات المعدلة
| الملف | التغيير |
|-------|---------|
| `bookJourneysController.js` | APIs للتعهد والرفقة والتقويم |
| `bookJourneys.js` (routes) | Routes جديدة |
| `bookJourneysService.js` | Service functions للـ Frontend |
| `BookJourneysPage.jsx` | Modal بدء الختمة مع التعهد |
| `JourneyDetailsPage.jsx` | عرض التعهد + BuddyCard + ProgressCalendar |

## 📱 تجربة المستخدم

### رحلة المستخدم الجديدة
1. **اختيار الكتاب** → يظهر Modal جميل
2. **تحديد الورد اليومي** → اختيارات سريعة أو إدخال يدوي
3. **كتابة التعهد** → نية القراءة والهدف
4. **البدء** → الختمة تبدأ!
5. **القراءة اليومية** → التعهد يظهر كتذكير
6. **إضافة رفيق** → تحفيز متبادل
7. **متابعة التقدم** → تقويم ملون + إحصائيات
